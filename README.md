# Clinic Queue Manager

## What this app does
This is a simple web application that helps manage patients in a clinic.
Patients are added to a queue and served in order (FIFO).

## Features
- Add patient
- View waiting list
- Serve next patient
- See total patients served
- Timestamp for each patient

## How to run
1. Install Python
2. Install Flask:
   pip install flask

3. Run the app:
   python app.py

4. Open browser:
   http://127.0.0.1:5000/
